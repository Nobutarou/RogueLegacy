# 金が無くて何もできない 

ゲーム初期に起きた私の体験。ゴールドを貯められずに死んでしまい、レベルアップも鍛冶屋もルー
ンも開放できない状態。

## 原因

入る部屋ごとに攻略しようとしてしまい、すぐに死んでしまう。

## 対策

城のボス部屋の宝箱を目指す。具体的に

* 避けられるなら戦わない
* 宝箱は狙わない
* 中ボスは戦わない

それから、奇妙なエルフの賭けには、必ず乗ること。期待値は 1.0 （もしゴールドアップの恩恵が
あれば 1.0 より大きい） であり、平均的に損はないことと、開放に必要なゴールドを持ち帰れない
なら意味がないことが理由
